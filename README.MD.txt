Локальный запуск
1. Открыть командную строку.
2. Перейти в папку со скриптом.
3. Запустить сервер командой "node server.js".
При необходимости установить/обновить библиотеки:
npm install express
npm install socket.io
4. После появления сообщения "server running at http://localhost:3000" перейти в браузере (протетировано в Chrome) на http://localhost:3000.
5. Ввести одно из слов:  html, javascript, python. Затем нажать кнопку "Send word".
